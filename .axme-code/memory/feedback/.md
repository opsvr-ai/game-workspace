# 改服务端代码后必须重启服务器

修改 apps/server/src/ 下任何 .ts 文件后，必须 fuser -k 3001/tcp && pnpm dev:server 重启。NestJS热重载不可靠，不能假设新代码已生效。每次改完→重启→curl验证→才算完成。
