# Conventional Commits + 自动文档更新

每次 feature/fix 提交后必须自动更新 CHANGELOG.md (Keep a Changelog 格式)、README.md (新端点/页面/功能/依赖变化)、docs/ (ARCHITECTURE.md/DEPLOYMENT.md/USER_MANUAL.md)，然后以 docs: update documentation for <feature> 提交文档变更。提交格式遵循 Conventional Commits: feat/fix/chore/docs/refactor。
