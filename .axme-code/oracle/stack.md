## Stack

- **语言:** TypeScript 5.5 (全栈)
- **运行时:** Node.js >= 18
- **包管理:** pnpm >= 8 (workspace monorepo)
- **前端:** React 18, Ant Design 5, React Router 6, Zustand 5, Vite 6, Recharts, Socket.IO Client
- **后端:** Nest.js 10, Prisma 5, Socket.IO 4, Passport JWT, bcryptjs, class-validator/transformer, multer
- **数据库:** PostgreSQL 16 (via Prisma ORM)
- **缓存:** Redis 7
- **部署:** PM2, Docker Compose
- **测试:** Vitest 4
- **桌面端:** Electron (companion-electron)
